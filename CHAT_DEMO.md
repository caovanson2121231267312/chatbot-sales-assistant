## 🔗 Cách Test

### Cách 1: Qua Web UI
- Mở http://localhost:8080
- Chat trực tiếp với bot

### Cách 2: Qua Terminal
```bash
docker exec -it rasa_server rasa shell
```

### Cách 3: Qua API
```bash
curl -X POST http://localhost:5005/webhooks/rest/webhook \
  -H "Content-Type: application/json" \
  -d '{"message": "xin chào", "sender": "user1"}'
```

## 📝 Ghi chú

- Bot hỗ trợ **NLP** (Natural Language Processing) nên có thể hiểu các câu hỏi với nhiều cách diễn đạt khác nhau
- Một số tính năng yêu cầu **Action Server** đang chạy
- Dữ liệu sản phẩm được lấy từ **MySQL Database**
